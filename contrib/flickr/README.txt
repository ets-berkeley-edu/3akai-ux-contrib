Flickr Widget:

To get the flickr widget working you need to add an api_key.
You can request an api_key by surfing to: 'http://www.flickr.com/services/apps/create/'.

To add the api key; 
Navigate to the Nakamura (open-experiments) via the terminal then do:

vim /load/org.sakaiproject.nakamura.proxy.ProxyClientServiceImpl-config.cfg

and add flickr_api_key=your api key 
(flickr_api_key=93a86c06dc382a19bff0d4d24872ecab)
